আপনার Wi-Fi যদি কেও চুরি করে চালায় তাহলে আপনি বুঝতে পারবেন কিভাবে?

এই সেটিংসটির মাধ্যমে আপনি আপনার Wi-Fi এ Alert অপশনটি অন করে রাখুন যেমন : প্রথমে আপনার Wi-Fi আপনার ফোনে কানেক্ট করুন তারপরে এখানে পাসওয়ার্ড এর জায়গায় পাসওয়ার্ড দিয়ে Tarn On Notification বাটনে ক্লিক করবেন তারপরে আপনাকে বলবে Successfully আপনার অনুরোধটি সক্রিয় হয়েছে

অথবা input > Password > Tarn on Notification > Click > Successfully >>= Done✅

Note: অবশ্যই সঠিক Password লিখে তারপরে Ok বাটনে ক্লিক করবেন তাহলে আপনার Settings চালু হবে।

এটি অন করে রাখলে যদি কেও অচেনা আপনার Wi-Fi চালায় সাথে সাথে আপনার ফোনে Notification আসবে আর তার সাথে সাথে আপনাকে Alert করে বা সতর্ক করে দিবে যে অন্যকেউ আপনার Wi-Fi ব্যাবহার করছে।
